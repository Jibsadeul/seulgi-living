## 1. 기능 정의

### 목적

자취 경험이 적은 청년들은 낯선 동네에서 필요한 편의시설을 매번 따로 검색해야 한다.
슬기로운 자취생활은 "자취생활에 필요한 것들을 하나의 앱에서 해결"하는 게 핵심 가치이기 때문에,
외부 앱으로 이탈하지 않고 앱 안에서 주변 시설을 바로 확인할 수 있어야 한다.

### 핵심 유스케이스

- 현재 위치 기반으로 주변 편의시설 7종을 지도에서 확인한다.
- 카테고리 원탭 필터로 원하는 시설만 빠르게 필터링한다.
- 마커 클릭 시 장소 정보를 확인하고, 카카오맵으로 이동해 상세 정보(길찾기·영업시간 등)를 확인한다.

## 2. 기술 결정

### 지도 구현 방식: 카카오맵 WebView

| 항목      | 내용                                                                                               |
| --------- | -------------------------------------------------------------------------------------------------- |
| 방식      | `react-native-webview` + 카카오맵 JS SDK                                                           |
| 선택 이유 | 시설 데이터·검색 UI를 카카오가 제공하므로 별도 구축 불필요. 구현 공수 대비 완성도 높음             |
| 기각      | `react-native-maps` + 카카오 로컬 API — 지도 렌더링만 제공, 나머지 전부 직접 구현해야 해 공수 과도 |

### 초기 로딩: 레이지 미리 마운트

- 앱 첫 진입 시 백그라운드에서 WebView를 미리 마운트
- 사용자가 지도 탭을 눌렀을 때 로딩이 이미 완료된 상태로 제공
- 스켈레톤 UI 별도 적용 없음 (미리 마운트 시 보여줄 타이밍 없음)

## 4. 핵심 기능

### 4-1. 카테고리 필터

- 지도 상단에 카테고리 칩 7종을 가로 스크롤로 표시
- 하나만 선택 가능 (토글 방식, 선택 해제 시 전체 마커 제거)
- 선택 시 카카오맵 JS에 카테고리 코드로 검색 요청 → 마커 렌더링

### 4-2. 자유 검색

- 카카오맵 내장 검색창을 그대로 활용
- 카테고리 외 시설도 자유롭게 검색 가능
- GPS 거부 시 동네 입력을 동일 검색창에서 처리 (재요청 없이 바로 동네 검색으로 대체)

### 4-3. 내 위치로 돌아가기

- 지도 위에 플로팅 아이콘 버튼으로 표시 (우하단 고정)
- 탭 시 현재 GPS 위치로 지도 재중심 이동
- `MOVE_TO_LOCATION` 메시지를 WebView로 전송해 처리

### 4-4. 마커 → 카카오맵 이동

- 마커 클릭 시 말풍선(인포윈도우)에 장소명 표시
- 말풍선 클릭 시 바텀시트 오픈
  - 표시 정보: 장소명, 주소, 카카오 API 응답에서 받을 수 있는 최대 정보
  - 카카오맵으로 이동 버튼 제공 (길찾기·영업시간 등 상세 정보는 카카오맵 앱에서 확인)

### 4-5. 이 지역에서 재검색

- 카테고리 검색·자유 검색 결과가 있는 상태에서, 사용자가 지도를 직접 움직인(드래그/줌) 이후에만 노출
  - 검색 결과가 없거나(0건), 검색을 아예 하지 않은 상태에서는 노출 안 함
  - 내 위치 이동, 검색 자체로 인한 지도 중심 이동(프로그램에 의한 이동)은 노출 트리거에서 제외 — 사용자가 직접 움직였을 때만 인정
- 위치: 카테고리 칩 바로 아래, 가로 중앙
- 탭 시 직전과 동일한 검색 조건(선택된 카테고리 코드 또는 입력했던 키워드)으로, 현재 지도에 보이는 범위(bounds) 기준 재검색 → 버튼 숨김
- 카테고리 검색과 자유 검색 모두 동일하게 적용

## 5. WebView ↔ React Native 메시지 인터페이스

모든 메시지는 아래 공통 봉투 형식을 따른다.

```typescript
{ type: string; payload?: Record<string, unknown> }
```

### RN → WebView

| type               | payload                                             | 설명                                                |
| ------------------ | --------------------------------------------------- | --------------------------------------------------- |
| `SEARCH_CATEGORY`  | `{ code: string \| null; keyword: string \| null }` | 카테고리 필터 선택 시 / 재검색 시 검색 요청         |
| `SEARCH_KEYWORD`   | `{ keyword: string }`                               | 자유 검색 / 재검색 시 검색 요청                     |
| `FOCUS_MARKER`     | `{ x: string; y: string }`                          | 결과 리스트 아이템 탭 시 해당 좌표로 지도 중심 이동 |
| `CLEAR_MARKERS`    | -                                                   | 마커 전체 제거                                      |
| `MOVE_TO_LOCATION` | `{ lat: number; lng: number }`                      | 내 위치로 지도 이동                                 |

### WebView → RN

| type                 | payload                                                                                                | 설명                                                                                                                                                |
| -------------------- | ------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MAP_READY`          | -                                                                                                      | 지도 로드 완료                                                                                                                                      |
| `MARKER_CLICK`       | `{ id, place_name, category_name, address_name, road_address_name, phone, place_url, x, y, distance }` | 마커 클릭 시 장소 정보 전달                                                                                                                         |
| `SEARCH_RESULT`      | `{ places: MapPlace[] }`                                                                               | 카테고리/키워드 검색 결과                                                                                                                           |
| `SEARCH_ZERO_RESULT` | `{ keyword: string }`                                                                                  | 검색 결과 0건                                                                                                                                       |
| `MAP_MOVED`          | -                                                                                                      | 사용자가 지도를 직접 움직임(드래그/줌) → "이 지역에서 재검색" 버튼 노출 트리거. 프로그램에 의한 이동(내 위치 이동, 검색 결과 자동 중심 이동)은 제외 |
| `MAP_ERROR`          | `{ message: string }`                                                                                  | 지도 로드 실패                                                                                                                                      |

> `MARKER_CLICK` payload가 바텀시트에 표시할 데이터 전부다.

## 6. 사용자 흐름

### GPS 허용

```
지도 탭 진입
  → 현재 위치 중심 지도 표시
  → 카테고리 필터 선택
  → 마커 클릭 → 말풍선 → 바텀시트
  → 카카오맵으로 이동
```

### GPS 거부

```
지도 탭 진입
  → 카카오맵 검색창 안내 표시
  → 동네 직접 입력
  → 해당 위치 중심 지도 표시
```

## 7. 예외 처리

| 상황                                          | 처리                                                                                               |
| --------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| WebView 로드 실패 (`onError` / `onHttpError`) | 하단 토스트 "지도를 불러오지 못했어요. 네트워크를 확인해주세요" + 재시도 버튼. 전체 에러 화면 없음 |
| 검색 결과 없음                                | 카카오맵 내부에서 자체 처리. 별도 빈 상태 UI 불필요                                                |
| GPS 권한 거부                                 | 전체 화면 차단 없이 검색창 안내로 대체. 사용자 의사 존중                                           |
