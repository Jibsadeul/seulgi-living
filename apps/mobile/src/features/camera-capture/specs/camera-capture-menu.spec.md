# camera-capture 메뉴 명세

## 목적

하단 네비게이션 중앙 카메라 버튼을 눌렀을 때, 페이지 이동 대신 선택 메뉴를 띄운다.

## 범위

### features/camera-capture

- 중앙 카메라 버튼을 눌렀을 때 선택 메뉴를 연다.
- 선택 메뉴에는 `영수증 촬영`, `식재료 촬영` 두 항목이 있다.
- 메뉴 바깥을 누르면 메뉴를 닫는다.
- 각 항목을 누르면 `/(stack)/camera`로 이동하며 선택 모드를 함께 전달한다.

### entities/camera

- 카메라 선택 모드 타입을 정의한다.
- 선택지 라벨과 모드 매핑을 재사용 가능한 상수로 제공한다.

## 검증 기준

- 중앙 카메라 버튼을 누르면 선택 메뉴가 열린다.
- 메뉴 바깥을 누르면 메뉴가 닫힌다.
- `영수증 촬영`과 `식재료 촬영`이 각각 올바른 모드로 전달된다.
