# 장보기 항목 액션 메뉴 명세

## 목적

장보기 항목의 점 세 개 버튼을 누르면 수정/삭제 옵션을 선택할 수 있는 액션 메뉴를 표시한다.

## 이번 작업 범위

- 액션 메뉴 바텀시트 UI 표시 및 닫기
- 수정/삭제 실제 동작은 후속 작업에서 구현한다.

## 데이터 흐름

- 점 세 개 버튼을 누르면 해당 항목의 액션 메뉴 바텀시트가 열린다.
- 선택된 항목의 이름을 메뉴 상단에 표시한다.
- 메뉴에는 수정, 삭제 두 가지 옵션을 제공한다.
- 수정/삭제를 탭하면 메뉴를 닫는다. (실제 동작은 추후 구현)
- backdrop 탭 또는 pan down으로 메뉴를 닫을 수 있다.

## UI

### 액션 메뉴 바텀시트

- 상단에 선택된 항목 이름 표시
- 수정 옵션: 연필 아이콘 + "수정" 텍스트
- 삭제 옵션: 휴지통 아이콘 + "삭제" 텍스트 (`point-100` 색상)
- 취소 버튼

## 이번 작업 제외 범위

- 수정 폼 표시 및 저장
- 삭제 API 호출
