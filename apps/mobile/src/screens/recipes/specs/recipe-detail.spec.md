# RecipeDetailScreen

> 레시피 카드 클릭 시 이동하는 상세 화면. Figma 노드 541-2384 기준.

## 기능 정의

- 레시피 상세 정보 표시 화면.
- 위에서 아래로:
  1. 헤더: ← "레시피 상세" + 북마크/공유 아이콘 (detail variant)
  2. 히어로 이미지 (대형 사진 placeholder)
  3. 레시피 제목 + 태그 칩
  4. 재료 및 미디어: "대표 이미지(Main)" / "완성 이미지(Finish)" 탭 + 사진 placeholder
  5. 필요한 재료: 인원 기준 + 재료 테이블 (재료명, 수량)
  6. 조리 순서: 번호 매긴 스텝 (텍스트 + 선택적 사진), "더보기" 접기/펼치기
- 인터랙션:
  - 북마크 아이콘 → 로컬 스크랩 토글
  - "조리 순서 더보기" → 펼치기/접기

## 데이터 흐름

- `entities/recipes` 미구현이므로 route param `id` 기반 로컬 mock 데이터 사용.

## 특이 사항

- 라우트 `app/(stack)/recipes/[id].tsx` 이미 존재.
- 실제 API 연동은 추후 별도 작업.
