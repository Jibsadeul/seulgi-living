# PRD (Product Requirements Document)

> 원문 기준. 수정 시 팀 합의 필요.

## 제품명

슬기로운 자취생활 (청년 자립 올인원 앱)

## 핵심 기능

| 탭 | 기능 | 비고 |
|----|------|------|
| 청년 정책 | 맞춤 정책 추천, 카테고리/마감임박 필터, 스크랩, 알림 | 외부 API |
| 레시피 | 검색/필터, 상세, 스크랩, 직접 등록 | 공공 + 유저 레시피 |
| 냉장고 | 재료 입력/수량 관리, OCR 자동 인식 | 로그인 전용 |
| 지도 | GPS 기반 편의시설, 카테고리 필터 | 카카오맵 |
| AI 채팅 | 자연어 레시피/정책 추천, 대화 맥락 유지 | Claude API |
| 마이페이지 | 프로필 수정, 스크랩 관리, 로그아웃/탈퇴 | 로그인 전용 |

## 타깃 사용자

1. 사회초년생 직장인 (25세, 서울 자취) — 레시피 + 정책
2. 취업 준비생 (28세, 고시원) — 저예산 레시피 + 취업 정책
3. 지방 상경 대학원생 (27세) — 편의시설 + 간편 레시피

## 비기능 요구사항

- 외부 API 로딩 중 Skeleton UI 필수
- 외부 API 실패 시 캐싱 데이터 또는 오류 안내
- GPS 좌표 서버 저장 금지
- React Native 기반 iOS/Android 최적화
- 개인 데이터: 인증된 사용자만 접근

## 예외 흐름

| 상황 | 처리 |
|------|------|
| 비로그인 스크랩 | 로그인 안내 |
| GPS 권한 거부 | 위치 직접 입력 |
| 외부 정책 API 실패 | 캐싱 데이터 / 오류 메시지 |
| OCR 실패 | 수동 입력 모드 전환 |
| 검색 결과 0건 | Empty State + 전체 보기 유도 |
